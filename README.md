<h1> Mini-GHC </h1>

<h2> Année : M2 </h2>
<h2> Langage : OCaml </h2>

<p>
L'objectif de ce projet est d'implémenter un optimiseur préservant le typage de μCore (un fragment du langage Core de GHC) appliqué à lui-même.

Il constitue le projet d'un cours de Programmation Fonctionnelle et Systèmes de types.
</p>
